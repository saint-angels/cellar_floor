# Mold Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** A 3-second mold block fully rings the clearing at worldgen for a snappy first session, spreads through dark passable space afterward, and gold odds move per-terrain (rock/soft 0.9, mold 0.1) so weak blocks are not a money printer.

**Architecture:** `TerrainType` gains `GoldChance` and `SpreadMinutes` (+ derived per-tick `SpreadChance`); `SimConfig.GoldChance` dies; `mineStep`'s completion roll reads the terrain captured before the floor flip. A new late tick step scans cells with `SpreadChance > 0` and converts one random passable, unlit, fauna-free 8-neighbor via SetTerrain. Gen gains a `crust` pass between fill and veins. Client untouched.

**Tech Stack:** Go stdlib + BurntSushi/toml; headless Playwright e2e.

**Spec:** `docs/superpowers/specs/2026-07-12-mold-design.md`

```go
// spreadStep grows spreading terrain (SpreadChance > 0) into random
// adjacent passable, unlit, unoccupied cells. Index order for determinism.
func (w *World) spreadStep() {
	for i, tr := range w.Terrain {
		tt := w.terrainAt(tr)
		if tt == nil || tt.SpreadChance <= 0 {
			continue
		}
		if w.RandFloat() >= tt.SpreadChance {
			continue
		}
		p := Point{X: i % w.Width, Y: i / w.Width}
		start := w.RandN(len(neighbors))
		for k := 0; k < len(neighbors); k++ {
			n := neighbors[(start+k)%len(neighbors)]
			q := Point{X: p.X + n.X, Y: p.Y + n.Y}
			if !w.InBounds(q) || !w.Passable(w.At(q)) || w.Lit(q) || w.FaunaAt(q) != nil {
				continue
			}
			w.SetTerrain(q, tr)
			break
		}
	}
}
```
