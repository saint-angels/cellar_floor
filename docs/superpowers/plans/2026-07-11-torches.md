# Torches Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Players spend colony gold to place torches; light gates where dwarves mine, darkness scares them home, and gold becomes a random drop from mined rock.

**Architecture:** Torches and a permanent campfire are entities of new data-defined types (`kind = "structure"`, new generic `light_radius` field). The sim keeps a derived `lit` bitfield recomputed when light sources spawn or die. A "species" to entity "type" rename lands first as its own mechanical commit. Gold terrain and gold-sense are deleted; finished rock cells roll a random gold drop. The client derives the same lit set from streamed entities (zero new light protocol), draws a darkness veil, and gains an arm-then-click torch placement button backed by a new `torch` ws intent.

**Tech Stack:** Go stdlib + gorilla/websocket server, TypeScript/Vite canvas client, headless Playwright for e2e.

**Spec:** `docs/superpowers/specs/2026-07-11-torches-design.md`

## Global Constraints

- Commit messages: one sentence, under 70 characters, no Claude attribution, no em or en dashes anywhere in text or code comments.
- TDD: write the failing test, see it fail, implement, see it pass, commit.
- When piping `go test` output, use `set -o pipefail` so failures are not masked.

- Never touch the user's live server (port 8080) or the canonical `world.json`/`players.json`. All e2e runs use an isolated port (8083) with a scratch data dir whose `sim.toml` has its own `save_path`.

```go
// RecomputeLight rebuilds the derived lit bitfield from living light
// sources. Called on load and whenever a light source spawns or dies.
func (w *World) RecomputeLight() {
	w.lit = make([]bool, w.Width*w.Height)
	for _, id := range w.SortedIDs() {
		e := w.Entities[id]
		if e.Dead {
			continue
		}
		s, ok := w.cfg.Types[e.Type]
		if !ok || s.LightRadius <= 0 {
			continue
		}
		r := s.LightRadius
		for y := maxInt(0, e.Pos.Y-r); y <= minInt(w.Height-1, e.Pos.Y+r); y++ {
			for x := maxInt(0, e.Pos.X-r); x <= minInt(w.Width-1, e.Pos.X+r); x++ {
				dx, dy := x-e.Pos.X, y-e.Pos.Y
				if dx*dx+dy*dy <= r*r {
					w.lit[y*w.Width+x] = true
				}
			}
		}
	}
}

func (w *World) Lit(p Point) bool {
	if w.lit == nil {
		return false
	}
	return w.lit[p.Y*w.Width+p.X]
}
```

Hook the triggers:
- End of `SetConfig`: `w.RecomputeLight()`.
- End of `Spawn`, before `return e`: `if s.LightRadius > 0 { w.RecomputeLight() }`.
- End of `kill`, before `return`: `if s.LightRadius > 0 { w.RecomputeLight() }`.
- `NewWorld` leaves `lit` nil; `gen.Generate` gets light via the campfire `Spawn`.

// darkStep sends a creature standing in an unlit cell toward the nearest
// living light source. In a world with no light at all it does nothing.
func (w *World) darkStep(e *Entity) bool {
	if w.Lit(e.Pos) {
		return false
	}
	var light *Entity
	bestD := 1 << 30
	for _, id := range w.SortedIDs() {
		c := w.Entities[id]
		if c.Dead {
			continue
		}
		s, ok := w.cfg.Types[c.Type]
		if !ok || s.LightRadius <= 0 {
			continue
		}
		if d := Dist(e.Pos, c.Pos); d < bestD {
			light, bestD = c, d
		}
	}
	if light == nil {
		return false
	}
	e.Action = "fleeing the dark"
	w.moveToward(e, light.Pos)
	return true
}
```

package server

import (
	"fmt"

	"cellarfloor/internal/sim"

// placeTorch validates and spawns a player-funded torch. Caller holds s.mu.
func (s *Server) placeTorch(token string, x, y int) PlayerMsg {
	pm := s.playerMsg(token)
	if pm.State != "alive" {
		pm.Error = "you need a living dwarf"
		return pm
	}
	if s.world.Gold < 1 {
		pm.Error = "not enough gold"
		return pm
	}
	p := sim.Point{X: x, Y: y}
	if !s.world.InBounds(p) || !sim.Passable(s.world.At(p)) {
		pm.Error = "can't place a torch there"
		return pm
	}
	for _, id := range s.world.SortedIDs() {
		e := s.world.Entities[id]
		if !e.Dead && e.Pos == p && s.cfg.Types[e.Type].Kind == "structure" {
			pm.Error = "something already stands there"
			return pm
		}
	}
	s.world.Gold--
	s.world.Spawn("torch", p)
	s.pending = append(s.pending, sim.Event{
		Tick: s.world.Tick, Type: "placed",
		Msg: fmt.Sprintf("%s placed a torch", s.players[token].Name),
	})
	return pm
}
```

private recomputeLight() {
  this.lit = new Array(this.width * this.height).fill(false);
  for (const e of this.entities.values()) {
    if (e.dead) continue;
    const r = this.types[e.s]?.lightRadius ?? 0;
    if (r <= 0) continue;
    for (let y = Math.max(0, e.y - r); y <= Math.min(this.height - 1, e.y + r); y++) {
      for (let x = Math.max(0, e.x - r); x <= Math.min(this.width - 1, e.x + r); x++) {
        if ((x - e.x) ** 2 + (y - e.y) ** 2 <= r * r) this.lit[y * this.width + x] = true;
      }
    }
  }
  this.lightVersion++;
}
